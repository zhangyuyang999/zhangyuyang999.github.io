Python 基础

1.变量和类型

  • python中的基本变量通过=直接进行赋值，不需要申明变量类型。
  • python中的基本类型分为数字类型和非数字类型，数字类型就是常用的int，float，bool，complex这些，非数字类型就是一些容器类型，例如str，list，set，tuple，dict。

2.标识符和类型转换

  • python中的变量命名是区分大小写的，python中习惯使用蛇形命名法，就是在两个单词之间加_来分词。
  • 变量的命名不可以和python中的关键词同名，会产生冲突。

    3.类型转换和运算符

  • 类型转换
  • 运算符

    4.输入和输出

  • python中通过print()函数来输出内容。可以使用一定的语法进行格式化输出。
  • 其中%s代表代表字符串 %d 代表整数 %f 代表浮点数
  • python中通过input()函数来输入内容，用户输入的任何内容python都认为是一个字符串。
  • 字符串变量 = input(“提示信息：”)，提示信息会打印到控制台

5.分支和循环

  • if else 语句格式
  • while 语句格式

6.容器数据类型

  • 1.list 列表 通过[ ]来表示列表，元素之间通过 , 来分割
  • 2.一个列表中的元素又是列表，那么就是列表的嵌套。
  • 3.列表常用操作
    • 列表.append(值) 在末尾追加数据
    • 列表.remove(值) 删除第一个出现的指定元素
    • 列表[索引]=值 修改指定索引处的数据
    • 列表[索引] 取指定锁定处的值
    • len() 列表长度 值 in 列表 判断列表中是否包含某值
    • 列表.sort() 对列表进行排序
  • if…in：判断某个元素是否在列表中，如果在，if的条件为True
  • for…in：从头到尾 依次从 列表 中取出 每一个元素
  • for循环里面没有遇到break语句，for执行完后，则会执行else的分支
  • 2.元组的定义
    • 元组用 () 定义
    • 也用于存储一组数据，元素之间使用 , 分隔
    • 元组的 索引(下标) 从 0 开始
    • 元组中 只包含一个元素 时，需要 在元素后面添加逗号
    • tuple（元组）与列表类似，不同之处在于元组的 元素不能修改
  • 3.字典的定义
    • 字典同样可以用来 存储多个数据
      • 通常用于存储 描述一个 物体 的相关信息
    • 字典用 {} 定义
    • 字典使用 键值对 存储数据，键值对之间使用 , 分隔
      • 键 key 是索引，值 value 是数据
      • 键 和 值 之间使用 : 分隔
      • 值 可以取任何数据类型，但 键 只能使用 字符串、数字或 元组
      • 键必须是唯一的
    • 字典的基本操作
      • 字典[键]=值 修改指定键的值
      • 字典.pop(键) 删除指定键值对，返回值
      • 字典[键]=值 修改指定键的值
      • 字典[键] 字典.get(键) 获取指定键的值
      • for key, value in 字典.items() 遍历字典获取所有键值对
